
README

1. Navigate to the src-file in the terminal
2. Make sure that the newest version of Bochs is installed
3. Write "make" in the terminal

(3.1) If error(s) occurs, write "make clean" and then "make" in the terminal

4. Write "bochs -q" in the terminal
5. A separate empty window should now appear 
6. Click on the terminal window and write "c" followed by "enter"
7. The bochs emulator window should now be active
